# EduBuddy - AI-Powered Study Companion

EduBuddy is an AI-powered platform designed to help students **study smarter, not harder**. It provides easy explanations, summarizes notes, generates quizzes and flashcards, and offers motivational support — all personalized to the student’s needs.

---

## Features

- **Explain Concepts Simply:** Paste text or topics, and get AI-generated explanations in simple language.
- **Summarize Notes:** Convert long notes or articles into concise, understandable summaries.
- **Quiz & Flashcards Generator:** Automatically create quizzes or flashcards from your study material.
- **Motivational Support:** Receive AI-generated motivational quotes and tips.
- **Personalized Learning:** Tailored explanations and quizzes based on your inputs.
- **Interactive Chatbot:** Ask EduBuddy questions in a conversational interface.

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite, TailwindCSS
- **Backend / AI:** Supabase Functions + OpenAI API (or any AI integration)
