## How It Works

1. Enter a **topic** or **study notes** into the text box.  
2. Choose what you want to do:
   - Simplify explanations  
   - Summarize notes  
   - Generate quizzes or flashcards  
3. The app uses your **AI API key** (e.g., OpenAI GPT-4) to generate results.  
4. View, copy, or download your learning content.
